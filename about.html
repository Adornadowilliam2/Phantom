The Phantom Troupe, also known as the Spider, is a notorious group of criminals in the Hunter x Hunter anime and manga series. In the series, the Phantom Troupe's website is briefly mentioned during the Yorknew City arc. The website serves as a platform for the group to showcase their activities and communicate with their members.

public class Pattern { public static void main(String[] args) { int n = 3; // Number of rows for (int i = 1; i <= n; i++) { // Print spaces before the numbers for (int j = 1; j <= n - i; j++) { System.out.print(" "); } // Print the numbers in increasing order for (int j = 1; j <= i; j++) { System.out.print(j); } // Print the numbers in decreasing order for (int j = i - 1; j >= 1; j--) { System.out.print(j); } // Move to the next line System.out.println(); } } }
